create github blog

1 Jan 2018 on Linux
Updated today

遇到的问题

  • 没找到setting
  • 没有先配置自己的用户名，导致github不认识你

建立一个博客项目

在github网站上新建一个项目，如下所示：

repository name必须是username.github.io格式，username就是你的github用户名

查看博客的网址

打开刚才新建的仓库，在仓库导航栏最右侧找到setting，并点击

进入页面之后，下来到底部

现在，就可以使用这个网址来访问你的博客了

安装jekyll

jekyll可以将纯文本转化为静态网站的博客。接下来介绍如何在ubuntu上安装jekyll。

先安装依赖包：

sudo apt-get install ruby-full build-essential zlib1g-dev

配置gem的安装目录：

在~/.bashrc文件内加入以下内容，

# Install Ruby Gems to ~/InstalledApplication/gems	安装目录可以自己更改
export GEM_HOME="$HOME/InstalledApplication/gems"    
export PATH="$HOME/InstalledApplication/gems/bin:$PATH"

安装jekyll

gem install jekyll bundler

建立本地博客

建立一个jekyll站点：

jekyll new myblog

myblog是一个目录，你可以自己决定在哪个目录下创建这个目录。然后进入这个刚创建的目录

构建站点并使其在本地服务器上可用

bundle exec jekyll serve

bundle exec 表示在当前项目依赖的上下文环境中执行命令 jekyll serve

然后就可以在http://localhost:4000上浏览这个博客了
